conan采集脚本
---

> 此脚本目的是在QDataStandard, QDataLite, QPlus 三种产品的机器上运行， 收集一些软硬件信息进行统计汇总，最后生成一张excel表格。

##  使用方法

- 将 `conan-{VERSION}.tar.gz` 包上传到节点任意位置解压

- vi 打开 `config.yml` 进行修改，确认后保存退出

- ./conan 运行脚本， 根据提示进行操作
